MyEclipse2014安装并建立java项目和web项目

MyEclipse安装直接全部next就行了

选择all

选择64位(根据jdk位数决定，要跟jdk的位数一样)

安装完打开，选择自己的工作目录

然后进入MyEclipse

这里新建java项目

选择java-project

起一个项目名称

然后直接finish就ok了

这里的是视图选择，随便点即可

这里简单的java项目就建立好了

看一下资源管理器中的文件目录

下面新建一个web项目

起一个项目名

直接finish即可，这里直接建立好了web项目

打开项目，这里默认生成了一个index.jsp的文件

这里可以直接修改内容的

这里的MyEclipse里面有内置的tomcat服务器，也可以使用自己配置的tomcat服务器

右键项目名，进行MyEclipse的服务器配置

这里选择tomcat7就行

然后点击ok就会启动服务，这里直接打印出了我们的index.jsp中的内容

在这里可以停止服务

或者直接使用浏览器打开也可以看到页面

这里的前缀就相当于localhost

更改为localhost依然可以访问
