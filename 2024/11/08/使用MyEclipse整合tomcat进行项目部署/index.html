使用MyEclipse整合tomcat进行项目部署

在实际的开发环境中，尽量不使用MyEclipse等工具自带的tomcat等服务，最好配置自己下载的服务。

打开MyEclipse的server配置，点击configure server

选择tomcat中我们下载的tomcat7版本将disable更改为enable

然后选择tomcat的安装目录

选择目录

选择目录完毕后点击jdk进行配置

如果这里没有自动选择jdk就点击add手动添加jdk

选择c盘的java目录下的jre文件目录

点击确定后jre下的jar包都会加载进来，直接finish即可

这里就配置好了我们自己的jdk

都配置好后，点击apply然后ok就行了

然后查看服务器就会多出一个我们自己配置的tomcat7.0服务器

正常开发环境中最好使用自己配置的tomcat等服务器，使用软件自带的服务器可能会出现bug等其他问题。

新建一个webproject项目测试一下自己配置的tomcat服务器

依然是起一个项目名

然后java-version选择1.7版本，服务器选择我们自己配置的tomcat服务器

然后直接next最后finish就行

项目建立好了之后，点击apache tomcat7.0 Runtime Libraries

这里可以看到tomcat的依赖就是我们自己配置的tomcat的安装路径

然后打开index.jsp文件修改一下java代码运行测试一下

然后鼠标点击项目，点击项目添加按钮，将项目add到里面

add之后这里依然是选择我们自己配置的tomcat服务器

项目部署后，tomcat会将我们的项目部署到webapps目录下

最后finish即可

然后就会出现我们刚刚部署的项目，点击ok

然后点击服务器，选择我们配置的tomcat服务器，然后start启动

启动后控制台会输入tomcat的启动日志，这里说明启动成功

打开浏览器访问localhost:8080,可以看到apache-tomcat已经启动

访问我们的项目，出现index.jsp的页面内容

第二种便捷部署项目和快捷启动tomcat服务的方法

先显示server选项卡，点击show view 之后选择server

在server中我们可以看到刚刚部署在我们自己tomcat服务器下的项目

这里右键项目可以快捷的将项目在tomcat中移除

右键tomcat服务器可以快捷的停止或者重启tomcat服务器，也可以快捷添加项目到tomcat服务器中

最后查看一下资源管理器中的项目位置
