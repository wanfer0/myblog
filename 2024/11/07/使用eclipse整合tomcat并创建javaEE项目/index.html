使用eclipse整合tomcat并创建javaEE项目

第一步要安装好java

第二步配置好java的环境变量

配置过程可以参考这里

配置path环境变量

配置完毕后使用cmd输入java以及java-version检查一下

检查完毕后下载tomcat

下载完毕后运行tomcat服务，如果运行成功，说明java已经安装成功，并且环境变量已经配置成功

运行startup.bat文件，启动服务

启动成功后浏览器输入localhost:8080查看tomcat服务

之后下载eclipse软件，直接解压即可

启动eclipse

新建一个项目文件夹，并设置为项目工作空间

进入eclipse

这里点击Windows选项卡点击showview

点击servers，这里没有server，新建一个server

选择apache

选择安装的apache7.0,这里的服务名默认即可

选择刚刚安装的本地的apache7.0的目录

next，finish即可

之后就配置好了apache-tomcat的服务

这里新建一个动态web项目测试

这里起一个项目名字，选择自己的apache-tomcat服务，

Dynamic web module version选择为2.5版本

next，finish即可

这里的一个空的web动态项目就新建好了

新建一个jsp文件测试一下

取名index.jsp

将index.jsp中的编码格式替换成utf-8的形式

全部替换

替换完毕

使用out.print输出文字试试

右键tomcat服务器，选择add and remove

将wanfer项目添加到tomcat服务器中

启动服务器

下面的控制台console中可以看到服务器启动日志

server中显示wanfer项目已经启动

打开浏览器访问项目地址测试

输入localhost:8080/wanfer

成功打印出内容

这样就搭建好了一个简单的javaEE项目了，下面是本地的项目文件
