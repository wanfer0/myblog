使用eclipse新建java项目并运行

new一个project

选择java-project

起个名直接finish就行了

这里就搭建好了一个java新项目

查看项目所在位置，右键项目点击properties

得到项目位置

打开项目位置，这里有一个bin目录，但是在eclipse中是隐藏的，里面存放class文件

新建一个class文件在src目录下

起一个class名称

这里已经生成了class文件在src目录下

这里来到资源管理器中发现src目录和bin目录下都会创建一个class文件

src目录下的class文件

bin目录下的class文件

写一个helloworld并打印出来

运行服务

这里一共三种运行方式：右键run或者左上角运行图标或者ctrl+f11都可以运行

运行后控制台打印出hello world
